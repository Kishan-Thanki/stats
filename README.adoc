= go-math-utils

*A collection of safe, overflow-proof mathematical utilities built in Go.*

image:https://img.shields.io/badge/License-MIT-blue.svg[License: MIT,link=https://github.com/kishan-thanki/go-math-utils/blob/main/LICENSE]

This package provides common mathematical functions — GCD, LCM, Factorial, Fibonacci, Power, IsPrime, PrimeFactors, and SumNaturalNumbers — all hardened against *integer overflows*, *division-by-zero*, and *infinite loops* that plague naïve implementations.

== Features

* *GCD*: Greatest Common Divisor via Euclidean algorithm. Handles negatives and `math.MinInt` safely (panics on overflow).
* *LCM*: Least Common Multiple with division-before-multiply to prevent premature overflow.
* *Factorial*: Overflow-protected with platform-aware limits (n≤20 on 64-bit, n≤12 on 32-bit).
* *IsPrime*: Optimized step-by-6 trial division using `i <= n/i` guard to avoid `i*i` overflow on large primes.
* *Fibonacci*: Safe up to the 92nd term on 64-bit platforms.
* *Power*: Logarithmic binary exponentiation — O(log b) instead of O(b).
* *PrimeFactors*: Overflow-safe factorization returning `([]int, error)`.
* *SumNaturalNumbers*: Uses the Gauss formula with overflow-safe half-term multiplication.
* *Numeric Helpers*: Generic, overflow-aware implementations of `Min`, `Max`, `Clamp`, `Abs`, `Sign`, `IsEven`, and `IsOdd` for generic types.
* *EulerTotient*: Euler's totient function phi(n) to count coprimes up to n.
* *Divisors*: Sorted slice of all positive divisors, fully overflow-safe.
* *NextPrime*: Smallest prime strictly greater than n, with overflow panic boundaries.
* *NthPrime*: The n-th prime number, with error checking on bounds/overflow.
* *PerfectNumber*: Checks if a number equals the sum of its positive proper divisors.
* *Slice Utilities*: Generic, overflow-safe slice operations including `SumSlice`, `MinSlice`, and `MaxSlice`.

== Performance

This library is designed for high-performance and high-throughput scenarios, focusing on two optimization pillars:

* *Zero Heap Allocations*: All core mathematical computations (e.g., `GCD`, `LCM`, `Fibonacci`, `IsPrime`) run entirely on the stack. They trigger zero heap allocations, preventing garbage collection (GC) pauses when executed inside tight application loops.
* *Algorithmic Efficiency*: Optimized computational strategies are used wherever possible (such as $O(\log b)$ binary exponentiation for `Power` and step-by-6 trial division for primality checks).

To view raw, verifiable benchmark results from a sample execution, see link:./benchmarks.txt[benchmarks.txt].

== Usage

[source,go]
----
package main

import (
	"fmt"

	"go-math-utils/integer"
	"go-math-utils/numeric"
	"go-math-utils/sequence"
	"go-math-utils/stats"
)

func main() {
	// GCD — returns generic type
	fmt.Println("GCD of int8(12) and int8(18):", integer.GCD(int8(12), int8(18))) // 6

	// LCM — returns generic type
	if lcmVal, err := integer.LCM(56, 98); err == nil {
		fmt.Println("LCM of 56 and 98:", lcmVal) // 392
	}

	// Factorial — generic type
	if factVal, err := integer.Factorial(int16(5)); err == nil {
		fmt.Println("Factorial of int16(5):", factVal) // 120
	}

	// IsPrime — returns bool
	fmt.Println("Is 29 prime?", integer.IsPrime(29)) // true

	// Fibonacci — returns generic type
	if fibVal, err := sequence.Fibonacci(uint32(6)); err == nil {
		fmt.Println("Fibonacci of uint32(6):", fibVal) // 8
	}

	// Power — returns generic type
	if powVal, err := integer.Power(2, 3); err == nil {
		fmt.Println("2 raised to the power of 3:", powVal) // 8
	}

	// PrimeFactors — returns generic slice
	if factors, err := integer.PrimeFactors(56); err == nil {
		fmt.Println("Prime factors of 56:", factors) // [2 2 2 7]
	}

	// SumNaturalNumbers — returns generic type
	if sumVal, err := sequence.SumNaturalNumbers(5); err == nil {
		fmt.Println("Sum of first 5 natural numbers:", sumVal) // 15
	}

	// EulerTotient — returns generic type
	if phi, err := integer.EulerTotient(10); err == nil {
		fmt.Println("Euler totient of 10:", phi) // 4
	}

	// Divisors — returns generic slice
	if divs, err := integer.Divisors(12); err == nil {
		fmt.Println("Divisors of 12:", divs) // [1 2 3 4 6 12]
	}

	// NextPrime — returns generic type (panics on overflow)
	fmt.Println("Next prime after 14:", integer.NextPrime(14)) // 17

	// NthPrime — returns generic type
	if prime, err := integer.NthPrime(5); err == nil {
		fmt.Println("5th prime:", prime) // 11
	}

	// PerfectNumber — returns bool
	fmt.Println("Is 28 perfect?", integer.PerfectNumber(28)) // true

	// Numeric helpers
	fmt.Println("Min of 10 and 20:", min(10, 20)) // 10

	// Stats slice calculations
	intSlice := []int{1, 3, 3, 2, 3, 4, 5}
	avg, _ := stats.Average(intSlice)
	fmt.Printf("Average of %v is %.2f\n", intSlice, avg) // 3.00

	sum, _ := stats.SumSlice(intSlice)
	min, _ := stats.MinSlice(intSlice)
	max, _ := stats.MaxSlice(intSlice)
	fmt.Printf("Sum: %d, Min: %d, Max: %d\n", sum, min, max) // 21, 1, 5
}
----

== API Reference

=== `integer` Package (Generic)

[cols="1,2,2", options="header"]
|===
| Function | Signature | Notes
| `GCD` | `GCD[T Integer](a, b T) T` | Always non-negative; panics on overflow (e.g. `MinInt`, `0`)
| `LCM` | `LCM[T Integer](a, b T) (T, error)` | Returns `(0, nil)` if either input is 0
| `Factorial` | `Factorial[T Integer](n T) (T, error)` | Error if n < 0 or result overflows bounds of T
| `IsPrime` | `IsPrime[T Integer](n T) bool` | Overflow-safe for primes near bounds of T
| `Power` | `Power[T Integer](a, b T) (T, error)` | Error if b < 0 or result overflows bounds of T
| `PrimeFactors` | `PrimeFactors[T Integer](n T) ([]T, error)` | Error if n ≤ 1
| `EulerTotient` | `EulerTotient[T Integer](n T) (T, error)` | Counts positive integers up to n that are coprime to n; error if n ≤ 0
| `Divisors` | `Divisors[T Integer](n T) ([]T, error)` | Returns sorted positive divisors of n; error if n == 0 or if divisors overflow T
| `NextPrime` | `NextPrime[T Integer](n T) T` | Smallest prime strictly greater than n; panics if next prime overflows T
| `NthPrime` | `NthPrime[T Integer](n T) (T, error)` | Returns the n-th prime; error if n ≤ 0 or if prime overflows T
| `PerfectNumber` | `PerfectNumber[T Integer](n T) bool` | Returns true if n equals the sum of its positive proper divisors
|===

=== `numeric` Package (Generic)

[cols="1,2,2", options="header"]
|===
| Function | Signature | Notes

| `Clamp` | `Clamp[T cmp.Ordered](val, min, max T) T` | Clamps val between min and max; panics if min > max
| `Abs` | `Abs[T SignedOrFloat](x T) T` | Absolute value; MinInt wraps due to overflow
| `Sign` | `Sign[T SignedOrFloat](x T) int` | Returns -1 (negative), 0 (zero), or 1 (positive)
| `IsEven` | `IsEven[T Integer](x T) bool` | Returns true if x is even
| `IsOdd` | `IsOdd[T Integer](x T) bool` | Returns true if x is odd
|===

=== `stats` Package (Generic)

[cols="1,2,2", options="header"]
|===
| Function | Signature | Notes
| `Average` | `Average[T IntegerOrFloat](slice []T) (float64, error)` | Returns the arithmetic mean; error if slice is empty
| `Median` | `Median[T IntegerOrFloat](slice []T) (float64, error)` | Returns the median; error if slice is empty
| `Mode` | `Mode[T cmp.Ordered](slice []T) (T, error)` | Returns the most frequent value; error if slice is empty
| `SumSlice` | `SumSlice[T IntegerOrFloat](slice []T) (T, error)` | Returns the sum of slice elements; error on empty or overflow
| `MinSlice` | `MinSlice[T cmp.Ordered](slice []T) (T, error)` | Returns the minimum element of a slice; error if empty
| `MaxSlice` | `MaxSlice[T cmp.Ordered](slice []T) (T, error)` | Returns the maximum element of a slice; error if empty
|===
